
سیستم مدیریت کلینیک - نسخه MVP واقعی

امکانات:
- ذخیره نوبت در دیتابیس
- ثبت مراجع
- ثبت وضعیت نوبت
- ثبت مبلغ پرداختی
- داشبورد مالی اولیه

اجرا:

1) نصب Python
https://python.org

2) نصب Flask

pip install flask

3) اجرای پروژه

python app.py

4) باز کردن مرورگر

http://127.0.0.1:5000

برای آیفون:
پروژه را داخل Replit آپلود کن و Run بزن.
